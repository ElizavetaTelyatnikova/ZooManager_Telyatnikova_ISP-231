# ZooManager
# Назначение программы
Программа представляет собой приложение для управления зоопарком. Она позволяет добавлять, удалять и обновлять зоопарки и животных, а также просматривать список всех существующих зверей и привязывать их к определённым зоопаркам.
# Реализованные функции
* Просмотр списка всех зарегистрированных зоопарков.
* Добавление нового зоопарка.
* Удаление существующего зоопарка.
* Обновление местоположения зоопарка.
* Просмотр всех животных в базе данных.
* Добавление новых животных.
* Удаление животного из базы данных.
* Изменение имени животного.
* Привязывание животного к выбранному зоопарку.
* Отвязывание животного от выбранного зоопарка.
# Запуск проекта
1. Убедиться, что установлен пакет .NET Framework версии не ниже той, которую требует проект.
2. Скачать файл с репозитория.
3. Открыть решение Visual Studio (.sln файл): Open Solution → Class17_18_Telyatnikova_ISP-231.sln.
4. Настроить подключение к БД в файле "Конфигурация".
5. Запустить приложение нажатием кнопки F5 или выбрать пункт меню Debug → Start Debugging.
